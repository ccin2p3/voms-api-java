The VOMS Java APIs

Java APIs to validate and request VOMS attribute certificates

Download this project as a .zip file Download this project as a tar.gz file

The VOMS API can be used for

  • validating attribute certificates (ACs) inside a proxy and reading the attributes (VOMS Fully Qualified Attribute Names (FQANs) or generic attributes)
  • contacting a VOMS server in order to get an AC and creating a proxy out of it
